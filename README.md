# Netflix Login Page Clone

A responsive Netflix-inspired login page built using HTML and CSS. This project focuses on creating a modern user interface with responsive design principles and clean frontend structure.

## Features

* Responsive design for mobile, tablet, and desktop devices
* Modern Netflix-inspired UI
* Clean and structured HTML & CSS
* Interactive login form design
* Background overlay and typography styling
* Cross-device compatibility

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries

## Learning Outcomes

Through this project, I improved my understanding of:

* Responsive web design
* CSS Flexbox layouts
* UI replication techniques
* Media queries
* Frontend structure and styling
* Cross-browser compatibility
